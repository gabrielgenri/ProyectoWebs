🚀 Ponte en marcha en 5 minutos

Puede poner en funcionamiento un nuevo sitio de Gatsby en su entorno de desarrollo local en 5 minutos con estos tres pasos:

  1. Cree un sitio Gatsby a partir de un sitio Gatsby Starter.

    Configura tu blog de Gatsby con un solo comando:

    # install the Gatsby CLI globally
npm install -g gatsby-cli

# create a new Gatsby site using the default starter
gatsby new my-blazing-fast-site
     

  2. Inicie el sitio en developmodo .

    A continuación, dirígete al directorio de tu nuevo sitio e inícialo:

    cd my-blazing-fast-site/
gatsby develop
     

  3. ¡Abre el código fuente y comienza a editar!

    Su sitio ahora se ejecuta en http://localhost:8000. Abra el my-blazing-fast-sitedirectorio en el editor de código que prefiera y edite src/pages/index.js. Guarde los cambios y el navegador se actualizará en tiempo real.

En este punto, ya tienes un sitio web de Gatsby completamente funcional. Para obtener información adicional sobre cómo puedes personalizar tu sitio de Gatsby, consulta nuestros complementos y el tutorial oficial .

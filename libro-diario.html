BackToTopFunction

Creamos el Botón en html, ubicado en partialsdentro de partials/head.hbs

<button onclick="backToTopFunction()" id="backToTop" class="footer__bttop is-visible" aria-label="Back to top" title="Back to top">
<svg width="20" height="20">
<use xlink:href="https://demo.getpublii.eu/themes/simple/v3/assets/svg/svg-map.svg#toparrow"></use>
</svg>
</button>

Posicionamos el Botón mediante css, ubicado en partialsdentro de modulo-banner.hbs

    right: 2rem;

    z-index: 999;
}

Damos acción al Botón con js

document.addEventListener('DOMContentLoaded',
()=>{const backToTopButton=document.getElementById('backToTop');
if(backToTopButton)
{
const backToTopScrollFunction=()=>
{
if(document.body.scrollTop>400||document.documentElement.scrollTop>400)
{
backToTopButton.classList.add('is-visible');
}
else{backToTopButton.classList.remove('is-visible');
}
};
const backToTopFunction=()=>
{
window.scrollTo({top:0,behavior:'smooth'});
};
window.addEventListener('scroll',backToTopScrollFunction);
backToTopButton.addEventListener('click',backToTopFunction);
}
}

La clase sectionsirve como componente para anidar los elementos que conforman los partialsdiseñados con modulo-(objeto).hbs y tienen sus correspondientes clases CSS con sus propiedades distintivas.

Definición del .background-modulo

Aquí defino las características del módulo-8objeto)y la forma en que se comporta el contenedor de la imagen de fondo. En las clases image-1, image-2 defino las fotografías que contendrá la section.

.background-modulo{
    height:60vh;
    /*height:100vh;*/
    width: 100%;
    background-position: center;
    background-repeat: no-repeat;
    background-size: cover;
   /*background-attachment: fixed;*/
   }
.image-1 {
  background-image: url("https://images.pexels.com/photos/2775196/pexels-photo-2775196.jpeg?auto=compress&cs=tinysrgb&dpr=2&h=750&w=1260")
}
.image-2 {
  background-image: url("https://images.pexels.com/photos/1532771/pexels-photo-1532771.jpeg?auto=compress&cs=tinysrgb&dpr=2&h=750&w=1260");
}
